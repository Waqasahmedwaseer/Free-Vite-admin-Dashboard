## Installation

**Prerequisites:** Node.js 18+

1. Install dependencies:
   `npm install`
2. (Optional) Create `.env.local` from `.env.example` and set `GEMINI_API_KEY`.
3. Start the dev server:
   `npm run dev`

## Scripts

- `npm run dev` - start dev server
- `npm run build` - production build
- `npm run preview` - preview production build

## Features

- Multipage admin layout with sidebar navigation
- Analytics, projects, tasks, billing, and support pages
- AI workspace with optional Gemini integration
- Responsive UI and dark mode
